# V5.7 Changelog — MENA additions from Home Centre playbook

## What this adds

Two purely additive entries pulled from the Home Centre visual-system playbook (Desert Elegance lane), adapted for nursery context. No removals, no rewording of existing data — just two new entries that fill the MENA cultural gap in the prompter.

### 1. New aesthetic — Gulf Villa Nursery

```
id:        gulfvilla
label:     Gulf Villa Nursery
mood:      Warm Gulf-villa calm, soft sand-beige plaster, honey oak warmth,
           gentle afternoon window light, MENA-rooted quiet luxury
materials: Soft limewash plaster walls in warm sand-beige, honey oak wide-plank
           floor, woven jute rug in oat, slim brass sconce wall-light, sheer
           linen drapery, single woven seagrass basket as accent (no arches,
           no architectural niches, no marble side tables, no console tables
           behind product)
language:  Gulf villa nursery, warm sand limewash, honey oak floor, woven jute,
           soft brass-warm afternoon light, MENA-rooted neutral
```

Notable: anti-directives baked into materials (no arches, no marble tables, no console tables) — defensive against the regressions we cleaned up in V5.3-5.6.

Added to `MECHANISM_SAFE_AES` so it's eligible for Bouncer / Walker / Swing / Baby Rocker / Recliner / Play Gym / Changing Table random rolls.

### 2. New Shot 8 wardrobe option — Modest-luxe MENA caregiver

```
id:    modluxe_mena
label: Modest-luxe MENA caregiver
desc:  Warm-toned modest-luxe MENA caregiver: oat or sand linen kaftan sleeve,
       optional silk-linen scarf draped softly, single delicate gold cuff or
```

Drops cleanly into the wardrobes pool. Adds explicit cultural representation as one option among seven on Shot 8 lifestyle shots.

## Deployment

### Step 1: Push `index_v57.html` to GitHub Pages

Activates Gulf Villa + MENA wardrobe in the local fallback `_A` and `_WD` arrays.

### Step 2: Run `V5.7_supabase_additions.sql` in Supabase SQL Editor

Two `INSERT … ON CONFLICT DO UPDATE` statements, plus verification SELECTs at the end. Idempotent — safe to re-run.

After both steps, both new entries appear in the live randomization rotation.

## What was deliberately skipped from the playbook

- **Kashida Heritage** — Kufic-style calligraphy art panels are IP-sensitive per the playbook; needs Landmark legal review on abstract-calligraphy guardrails. Hold.
- **Levantine Heirloom** — overlaps too much with the Luxe Neutral aesthetic we just removed in V5.6 (alabaster lamp, brass sconce, marble accents). Risks the same regression.
- **Mashrabiya dappled lighting** — too dramatic / editorial for current PDP direction. Save for campaign work if needed later.
- **Tier-2 "Warm Plaster" / "Soft Sand"** — V5 is single-tier; would add complexity without much new visual ground vs Gulf Villa.
